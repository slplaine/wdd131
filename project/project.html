My First Crochet Project

This is a simple granny square, perfect for beginners. It uses basic stitches and can be combined to create blankets or scarves.

Steps

  1. Start with a magic ring.
  2. Make 3 chains and 2 double crochets.
  3. Continue with sets of 3 double crochets separated by chains.
  4. Close the round and repeat.
